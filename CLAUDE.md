# Project Rules

## Autonomy
- NEVER ask for confirmation before taking any action
- NEVER ask "should I proceed?", "is this OK?", "confirm?"
- Make decisions independently and execute immediately
- If unclear, choose the most reasonable option and do it
- User checks results after — not before

## Workflow
- Always run commands directly (never show code blocks for user to copy)
- Always git add + commit + push after any code change
- Fix errors end-to-end without stopping
